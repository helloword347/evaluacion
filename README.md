Ejemplo de una aplicacion que permite ver ubicacion, tomar o subir archivos asi como los que estan asignados, ademas de contar con un login para la verificacion de los usuarios
primero que nada se crearon dos archivos la API esta en la carpeta "principal_evaluacion", mientras que los archivos del Flutter estna en la carpeta "evaluacion_3"
primero agregamos la base de datos (paquexpress) a Xampp
ahora ingresaremos al archivo main.py con la dirrecion de principal_evaluacion/main.py, este lo ejecutarenmos en Visula Code con el comando uvicorn main:app --reload   
despues se ejecutara el archivo main.dart de la carpeta evaluacion_3/lib, en visual code con el comando flutter run
y listo a experimentar
